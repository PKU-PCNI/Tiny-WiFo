# Tiny-WiFo
This simulation code package is mainly used to reproduce the results of the following paper [1]:

[1] H. Zhang, S. Gao,and X. Cheng, "Tiny-WiFo: A Lightweight Wireless Foundation Model for Channel Prediction via Multi-Component Adaptive Knowledge Distillation," in IEEE Wireless Communications Letters.

If you use this simulation code package in any way, please cite the original paper [1] above.
